# 常见问题 | FAQ

## 基础问题

### Q1: 数据从哪里来？

A: 数据来源于中华人民共和国国家统计局公开数据。

### Q2: 更新频率是？

A: 国家统计局每月15-17日发布上月数据。

### Q3: 为什么叫70城？

A: 是指国家统计局选取的70个大中城市进行房价监测。

## 使用问题

### Q4: 提取数据为空怎么办？

A: 请检查：
1. 参数是否正确
2. 城市名称是否正确
3. 时间范围是否正确

使用 list-cities 和 list-dates 查看可用选项。

### Q5: 图表显示乱码怎么办？

A: 确保系统已安装中文字体（思源黑体、微软雅黑等）。

### Q6: 支持导出Excel吗？

A: 使用 --output filename.xlsx 参数。

### Q7: 如何批量导出？

A: 使用shell脚本循环处理多个城市。

## 技术问题

### Q8: 支持Python调用吗？

A: 可以导入脚本模块使用。

### Q9: 如何设置代理？

A: 通过环境变量设置 HTTP_PROXY 和 HTTPS_PROXY。

## 故障排除

### Q10: 更新数据失败

A: 检查网络连接和URL格式。

### Q11: 校验失败

A: 运行 python scripts/validate_price.py --verbose 查看详细错误。
