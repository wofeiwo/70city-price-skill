# 数据参考 | Data Reference

本文档提供70城房价数据的详细结构说明。

## 数据概述

### 数据来源

中华人民共和国国家统计局发布的《70个大中城市商品住宅销售价格变动情况》。

### 时间范围

2006年1月至今，约4.5万条月度记录。

## 数据结构

### CSV列定义

| 列名 | 说明 | 示例 |
|------|------|------|
| DATE | 数据日期 | 2025/12/1 |
| ADCODE | 城市行政区划代码 | 110100 |
| CITY | 城市名称 | 北京 |
| FixedBase | 指数类型 | 同比/环比/定基比 |
| CommodityHouseIDX | 新建商品住宅指数 | 97.6 |
| SecondHandIDX | 二手住宅指数 | 91.5 |

## 指数类型说明

### 同比（Year-over-Year）

与上年同月相比：
- >100：价格上涨
- <100：价格下跌

### 环比（Month-over-Month）

与上月相比：
- >100：价格上涨
- <100：价格下跌

### 定基比（Fixed Base）

与固定基期相比，注意基期轮换问题。

## 数据质量

### 覆盖范围

- 时间覆盖：2006年1月至今
- 城市覆盖：70个大中城市
- 指数覆盖：同比、环比

### 已知限制

1. 2011年前后数据结构有变化
2. 2023年起官方不再公布定基指数
3. 1月份数据无年度平均列
